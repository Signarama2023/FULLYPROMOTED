        Email setup — one-time, ~2 minutes
        

          To turn on auto-email from a GitHub-hosted copy, edit the CONFIG block near the top of this file's script, then commit:
          
    
            
  1. Go to web3forms.com and create a free access key using your own email (you'll get a copy of every job).
    2. 
            
  2. Paste the key into WEB3FORMS_ACCESS_KEY.
    3. 
            
  3. Put your team's real addresses in TEAM_EMAILS, comma-separated — they're CC'd on every send.
    4. 
            
  4. Commit & push. Hit Send test above to confirm it lands.
    5. 
          

          The page is public, so the key is visible in the source — that's normal for Web3Forms. If it ever gets abused, regenerate the key or put the page behind a login (e.g. Cloudflare Access).
